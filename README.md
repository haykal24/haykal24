## 🏃 Indonesia Marathon

**Indonesia Marathon** is a digital ecosystem and authority platform for the running community in Indonesia.

The platform connects **runners**, **event organizers**, and **brands** in one growing ecosystem.

For runners, Indonesia Marathon provides:

- Running calendar
- Marathon schedule
- Race discovery
- Running guides
- Educational running content
- Event recommendations

For event organizers and brands, Indonesia Marathon provides:

- Targeted audience visibility
- Event submission
- Promotional campaign support
- Running community exposure
- Premium partnership opportunities

### Tech Highlights

- Backend: **Laravel 12**
- Frontend: **Nuxt 4 SSR**
- Architecture: **Headless API**
- Rendering: **Hybrid rendering with SWR**
- Asset Management: **Spatie Media Library**
- Cloud Storage: **Cloudflare R2**
- Image Optimization: **WebP/AVIF auto conversion**
- Performance: **Zero CLS optimization**
- API Optimization: **Separate listing and detail payloads to reduce JSON bandwidth**

## 🎟️ Nusatix

**Nusatix** is an end-to-end ticketing platform designed to replace manual event registration workflows such as Google Forms.

The platform helps event organizers manage the entire event process in one integrated system.

Core features:

- Event registration
- Online payment flow
- Ticket generation
- Participant management
- Ticket validation
- Check-in system
- Event operation support

The goal of Nusatix is to make event management simpler, faster, and more scalable for organizers.
